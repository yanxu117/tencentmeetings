# Tencent Meetings Repository

This repository stores meeting recordings, class notes, and educational materials from Tencent Meetings (腾讯会议).

## Structure

```
tencentmeetings/
├── README.md              # This file
├── 2026/
│   ├── 02-22/       # Date-organized folders
│   │   ├── class_2026-02-22.md  # Class recordings/notes
│   │   └── transcripts/          # Raw transcripts
│   └── ...
└── resources/            # Shared resources, materials
```

## Categories

### Meetings/Classes
- Organized by date (YYYY/MM/DD)
- Include class topics, key learnings, homework
- Format: `class_YYYY-MM-DD.md`

### Transcripts
- Raw conversation transcripts
- Format: `transcript_YYYY-MM-DD.txt`

### Resources
- Phrase lists, vocabulary sheets
- IELTS preparation materials
- Study guides and tips
